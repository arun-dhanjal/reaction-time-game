# Reaction Time Game

A simple browser-based game built with HTML, CSS, and JavaScript.

## How it works
- Click "Start Game"
- Wait for a random button to turn green
- Click it as fast as possible
- Your reaction time is recorded

## Features
- Randomised reaction delay
- Best / last score tracking
- Simple state handling using JavaScript

## What I learned
- setTimeout and timing logic
- DOM manipulation
- Event handling
- Basic game state management

## Notes
This was a guided learning project while practicing vanilla JavaScript. AI assistant (ChatGPT) was used to steer things in the right direction when I was unsure of certain methodologies and logic (in particular, setting the timer, randomly changing the colour of one of the buttons, and recording the reaction time).
